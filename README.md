Website portofolio pribadi untuk menampilkan karya, CV, dan informasi kontak.  
Dibangun dengan HTML, CSS, dan JavaScript, lalu di-*deploy* menggunakan [Vercel](https://vercel.app).

# Run and deploy 

This contains everything you need to run your app locally.

View your app : https://hadi-suwarno.vercel.app

## 📖 Fitur Utama
- Halaman utama dengan perkenalan singkat: *"Hi, I’m Hadi Suwarno"*
- Tombol **View My Work** untuk melihat proyek
- Tombol **Download CV** untuk mengunduh CV ATS-friendly
- Ilustrasi profesional yang menampilkan aktivitas coding
- Responsif untuk desktop dan mobile

## ⚙️ Teknologi
- HTML5, CSS3, JavaScript
- Bootstrap untuk styling responsif
- Vercel untuk hosting dan deployment

## 📂 Struktur Repository
- `index.html` → Halaman utama
- `style.css` → Styling
- `script.js` → Interaksi dasar
- `.gitattributes` → Konfigurasi GitHub
- `README.md` → Dokumentasi proyek

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
